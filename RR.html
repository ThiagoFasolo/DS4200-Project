What is the Sharpe Ratio?

Maybe include information about the Sharpe ratio... something like: (Fund Return - Risk-Free Rate) / Standard Deviation

It measures how much excess return an investor receives for the extra volatility they endure by holding a risky asset. A higher Sharpe ratio indicates better risk-adjusted performance. Generally:

  • Below 1.0 - Suboptimal risk-adjusted returns
  • 1.0 to 2.0 - Good risk-adjusted returns
  • Above 2.0 - Excellent risk-adjusted returns
  • Above 3.0 - Exceptional risk-adjusted returns (rare)
